# Operating System Projects/Assignments

These are attempts at learning how fundamental concepts of OS works.
These projects are completed during the semester but are **CURRENTLY UNDER
REFACTOR/OPTIMIZATION** and should only be use for reference.  

| Project Name | % done | Concepts Covered |
| ------------ |:------:| :--------------: |
| Shell        |    85% | fork, pids, exec, string processing, backend CLI |
| Concurrency  |    60% | semaphores, priority processing |
| FAT32        |    20% | file systems (access, modify), safe data processing |
| Heap         |    95% | memory allocation (schemes), handles and management |
